Pour toute questions, demande de nouvelle fonctionnalité, retour de bug, ... il est possible de me contacter soit par MP sur le forum, ou en se rendant sur topic de l'application sur le forum.

Accès au code
 
Le code de l'application est opensource, il est possible d'y accéder en allant sur le site de l'application sur github où vous pouvez accéder au détail de l'implementation, faire une branche alternative, soumettre des mise à jour, ...
  • Site de l'application
 
Confidentialité et vie privée
 
Comme vous pouvez le vérifier dans le code de l'application (voir ci-dessus), aucune information relative au mot de passe n'est stocké au sein de l'application. La gestion de l'authentification est réalisée à l'aide d'un cookie fourni par le site forum.hardware.fr. Le pseudo de l'utilisateur sera par contre enregistré de façon non encrypté dans l'application dans un fichier texte, ceci dans le but d'afficher le nom de l'utilisateur dans la page de réglage. Ce fichier sera détruit lors de la déconnexion. La configuration de l'application est également stockée dans un fichier texte dans le dossier de l'application. Les avatars des autres utilisateurs, sont eux aussi mis en cache dans le dossier de l'application. Aucune autre information ne sera stocké par l'application.
